# Tests for `data/abc`

This folder contains small manual test functions you can run in-game to verify behavior.

How to run

- Copy the `data/` folder into your world's `datapacks/`, run `/reload`, then run the functions below in-game, e.g. `/function abc:tests/test_give`.

Files:

- `test_give.mcfunction` — gives the executing player a test item and prints a message.
- `test_load.mcfunction` — simple message used to verify `load.mcfunction` behaviour.
