# Support

If you need help using this template or running the example functions, the best place to get help is:

- Open an issue using the appropriate issue template (bug or feature request).
- For quick questions, include the Minecraft version and a clear description of what you tried and what happened.

Community channels

- Consider joining Minecraft modding/dev communities (Discord, Modrinth forums) for real-time help.

When reporting a bug

- Provide steps to reproduce, the Minecraft version, and any relevant data from `/debug` or server logs.
